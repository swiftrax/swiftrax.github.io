Custom Designed

Final Product First

The Layout

I wanted to use the encoder and arrow keys from the Cannon Keys Satisfaction 75. I also wanted to have and F Cluster like a lot of old vintage keyboards. The reason for not having an OLED screen was because this was my first time designing a keyboard in KiCad and thought it would be less of a hassle without it. The backspace is shifted down one row because that is how my Arisu is set up. I also decided to go WKL because why not.

The Process

Once the layout was made in Keyboard Layout Editor you can paste the raw data into xyz's tool to make the KiCad schematic and PCB with the switches and diodes already in a matrix. I decided to go against placing the MCU and others on the board because I suck at SMD soldering. Instead I decided to let an Elite-C be the brains of this build. This was chosen over the normal pro micro because it has more i/o pins. My matrix alone required 22 of those pins. This left just enough to connect the encoder to it. Sadly this build has no RGB lights or anything fancy. Adding the micro controller and wiring it up was easy and I used ai03's guide to get used to KiCad before I did this project.

I designed the plates in Fusion360 given a DXF from this handy tool. I just added holes in the middle and aligned them all up to make sure all the holes would connect with the layers.

Manufacturing

I then added my own silks and got the PCBs sent to JLCPCB. They were able to finish manufacturing the boards in less than a week. They are fairly inexpensive for small batch of boards. The largest of the ones I ordered were about 373mm x 125mm.

Firmware

I wrote the firmware using QMK as it is the most used for mechanical keyboards. It took a bit of trial and error but I was eventually able to get the board's firmware to compile properly. I decided to add some simple macros to fill in the F Cluster on the left side. 


bool process_record_user(uint16_t keycode, 
keyrecord_t *record) {
  switch (keycode) {
	  case TASK:
      if (record->event.pressed) { //open task manager
        // when keycode TASK is pressed
        register_code(KC_LSFT);
		register_code(KC_LCTL);
		register_code(KC_ESC);
      } else {
        // when keycode TASK is released
		unregister_code(KC_LSFT);
		unregister_code(KC_LCTL);
		unregister_code(KC_ESC);
      }
      break;
  }
  return true;
};

Other macros were made in a similar way.
